- **Multiple Games**: Endless runner, chess, car racing, puzzle challenges, and more.

## 🌈 Color Palette

| Theme        | Hex Code | Usage                          |
|--------------|----------|--------------------------------|
| Midnight Blue | `#0A192F` | Backgrounds, dark UI panels   |
| Neon Cyan     | `#00FFFF` | Highlights, buttons, power-ups|
| Electric Purple | `#9D4EDD` | Titles, accents, glowing effects |
| Sunset Orange | `#FF6B35` | Alerts, score counters, energy bars |
| Soft White    | `#F8F9FA` | Text, icons, clean overlays   |

> **Tip:** Use gradients like `linear-gradient(90deg, #00FFFF, #9D4EDD)` for premium button effects.

---
